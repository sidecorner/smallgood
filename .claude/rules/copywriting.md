# Copywriting

## トーンの定義

**ひとことで言うと:** 深夜に話しかけてくれる、優しくて正直な友人

- 敬語を使わない（「ください」「ましょう」より「してみて」「いいよ」）
- 前向き強制をしない（「頑張ろう！」「今日もファイト！」は禁止）
- 労いを先に置く（書かせようとする前に、今日を認める）
- 短く。詩的に。改行を使う
- 「〜なくていい」という免罪符の構文を積極的に使う

---

## 禁止フレーズ

```
❌ 頑張ろう / ファイト / ガンバレ
❌ また明日も書いてね
❌ 今日もサボっちゃった？
❌ 〇日連続！すごい！
❌ 目標まであと◯日
❌ 完璧な一日だったね
❌ もっと書いてみよう
❌ 〜しなければならない / 〜すべき
```

## 承認済みコピープール

### 時間帯別挨拶（HomeScreen）
```js
hour < 5  → "夜更かしだね。お疲れさま。"
hour < 11 → "おはよう。今日もここにいるよ。"
hour < 17 → "こんにちは。ひと息ついてね。"
hour < 21 → "お疲れさまでした。"
hour >= 21 → "今日も一日、よく頑張ったね。"
```

### ジェントルトースト通知（GENTLE）
```
今日も一日お疲れさまでした。\n小さないいこと、ひとつだけ覚えてる？
書けなくていいよ。スタンプひとつだけでも十分。
頑張らなくていい。ただここに、あなたがいる。
ちいさなことでいい。今日、何かひとつよかったことは？
あなたの今日を、少しだけ教えてほしいな。
完璧じゃなくていい。そのままのあなたでいい。
```

### 記録完了メッセージ（DONE_MSG）
```
今日もよく頑張ったね ✨
小さないいことを見つけてくれてありがとう 🌸
あなたは十分すごい。本当に。
今日のあなたを、ちゃんと残せたね 🌙
こうして続けていることが、もうすてき 🌿
書いてくれて、よかった 💙
```

### 記録ヒント（EntryScreen上部）
```
小さなことでいい ✨
今日もここにいてくれてありがとう 🌿
あなたの言葉を待ってるよ 🌸
ゆっくりでいいよ 🌙
```

### プレースホルダー（入力欄の例示）
```
Set A: 朝のコーヒーが美味しかった / 電車で席を譲れた / 好きな曲がかかった
Set B: ランチが思ったよりおいしかった / 同僚に笑顔で挨拶できた / 空がきれいだった
Set C: 帰り道に夕焼けを見た / お風呂がゆっくり入れた / ちゃんと眠れそう
```

### CTA前の免罪符テキスト
```
（今日未記録）: "小さなことでいい"  /  初回: "書けたら、書いてみて"
（今日記録済）: "今日はもう書いたね ✦"
```

### 花のステージキャプション（HomeScreen）
```
0件 → ここから、はじまる
1〜2 → 芽が出たよ
3〜6 → ちゃんと育ってる
7〜14 → 続けてくれてありがとう
15〜29 → きれいに咲いてる
30+ → こんなに育ったね
```

## コピーを追加するときのチェック

1. 「頑張れ」に近いニュアンスがないか
2. 「続けなきゃ」という義務感を生まないか
3. 「まだ足りない」と感じさせる比較がないか
4. 敬語・丁寧語が強すぎて距離感が出ていないか
5. 一文が長すぎないか（目安: 24字以内）
