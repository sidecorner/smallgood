# Small Good — CLAUDE.md

> 頑張らない私が、少しだけ好きになる  
> 自己肯定感の低い働く女性（20〜30代）のための、3行の記録アプリ。

---

## このプロジェクトについて

**コンセプト:** 完璧主義傾向のあるターゲットが、罪悪感なく続けられる日記アプリ。  
カレンダーの空白を見せず、書いた分だけ花が育つ「加点方式」の体験設計が中心。

**主要成果物:** `small-good-app.jsx` — 単一ファイルのReactプロトタイプ

## クイックリファレンス

```
ターゲット   : 完璧主義傾向・自己肯定感低め・働く女性 20〜30代
記録形式     : 最大3行 × 50字 ＋ 気分スタンプ（任意）
成長ビジュアル: 花のみ（瓶・カレンダーなし）
花びらの色   : 記録時の mood カラーを直接反映
加点原則     : 減点なし・空白日は存在しない・数値は最小限
```
